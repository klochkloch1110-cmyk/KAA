# Edge Functions AVL 84

Папка зарезервирована под серверные функции Supabase.

## Планируемые функции

### `generate-order-number`

Серверная генерация номера заявки.

### `generate-report`

Генерация CSV/XLSX-отчетов.

### `run-ocr`

Интеграция с Yandex OCR. В первом релизе может быть заглушкой.

### `send-notification`

Создание уведомлений и отправка push через FCM на следующем этапе.

## Принцип

Все операции, влияющие на нумерацию, отчеты, OCR и уведомления, не должны быть реализованы только на клиенте.
