# run-ocr

Назначение: OCR-проверка фото ТТН.

Первый рабочий релиз:
- функция может быть не реализована;
- рейсы сохраняются с `ocr_status = 'not_required'` или `pending`.

После подключения:
- отправить изображение в Yandex OCR;
- сохранить сырой текст;
- извлечь номер ТТН и объем;
- сравнить с ручным вводом;
- обновить `trips.ocr_status`;
- создать запись в `ocr_results`.
