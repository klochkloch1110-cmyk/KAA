# send-notification

Назначение: создание уведомлений и отправка push.

Первый рабочий релиз:
- достаточно создавать записи в таблице `notifications`.

Следующий этап:
- подключить Firebase Cloud Messaging;
- отправлять push водителю при назначении заявки;
- отправлять push админу при новом рейсе или OCR-расхождении.
