# Прототипы AVL 84

В этой папке хранятся интерактивные и визуальные прототипы проекта.

## Текущие материалы

### `figma-make/`

Кодовая выгрузка из Figma Make: React/Vite-прототип интерфейса AVL 84.

Назначение:
- смотреть визуальное направление приложения;
- проверять расположение экранов и сценариев;
- использовать как источник UI-идей при дальнейшей реализации;
- не смешивать прототипный код с основной будущей кодовой базой.

Подробнее: [figma-make/README.md](figma-make/README.md)
