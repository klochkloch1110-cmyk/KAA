# AVL 84 - прототип из Figma Make

Это отдельный React/Vite-код, выгруженный из Figma Make по дизайну приложения AVL 84.

Оригинальный Figma-проект:
https://www.figma.com/design/gijCsfYKdyGTATp1uYvY2t/%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5-%D0%BF%D1%80%D0%B8%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D1%8F-%D0%BF%D0%BE-%D0%A2%D0%97

## Назначение папки

Папка `prototypes/figma-make` хранит только визуальный прототип и материалы Figma Make. Это не основная кодовая база будущего приложения.

Использовать как:
- визуальный референс UI;
- источник экранов, компонентов и стилевых решений;
- материал для последующей адаптации в рабочую архитектуру приложения.

## Структура

- `src/app/App.tsx` - главный экран прототипа и переключение между панелью руководителя и приложением водителя.
- `src/app/components/` - экраны и компоненты прототипа.
- `src/app/components/ui/` - сгенерированные UI-компоненты.
- `src/imports/` - изображения и вставленные материалы из Figma Make.
- `src/styles/` - стили, тема, Tailwind-настройки.
- `public/` - публичные ассеты прототипа.

## Запуск прототипа

Из этой папки:

```bash
npm i
npm run dev
```

Или с pnpm, если он установлен:

```bash
pnpm i
pnpm dev
```

## Важное замечание

Код Figma Make обычно полезен как быстрый интерактивный макет, но его не стоит напрямую считать финальной архитектурой продукта. Для боевой разработки нужно отдельно привести код к выбранному стеку, структуре модулей, бизнес-логике, авторизации, данным и API.
